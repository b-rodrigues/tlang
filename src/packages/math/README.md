# math

Pure numerical primitives.

## Functions

| Function | Description |
|----------|-------------|
| `sqrt(x)` | Square root |
| `abs(x)` | Absolute value |
| `log(x)` | Natural logarithm |
| `exp(x)` | Exponential (e^x) |
| `pow(base, exp)` | Exponentiation |

## Examples

```t
sqrt(16)      -- 4.0
abs(-5)       -- 5
log(1)        -- 0.0
exp(1)        -- 2.71828...
pow(2, 10)    -- 1024.0
```
