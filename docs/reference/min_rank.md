# min_rank

Minimum Rank

Assigns the minimum rank to ties, leaving gaps in ranks.

## Parameters

- **x** (`Vector`): The input vector.

## Returns

The ranks.

## Examples

```t
min_rank([1, 2, 2, 4])
-- Returns: [1, 2, 2, 4]
```
