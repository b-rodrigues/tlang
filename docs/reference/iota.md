# iota

Create a vector of ones

Creates a vector of length n filled with 1.0. Use with `seq` or `ndarray` for more complex interactions.

## Parameters

- **n** (`Int`): The length of the vector.

## Returns

A vector of ones.

## Examples

```t
iota(3)
-- Returns: [1.0, 1.0, 1.0]
```
