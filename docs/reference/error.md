# error

Raise Error

Raises a runtime error with a message and optional code.

## Parameters

- **message** (`String`): The error message.
- **code** (`String`): (Optional) Error code (e.g., "ValueError").

## Returns



## Examples

```t
error("Invalid input")
error("ValueError", "Must be positive")
```
