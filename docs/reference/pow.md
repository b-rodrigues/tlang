# pow

Power function

Calculates base raised to the power of exponent.

## Parameters

- **base** (`Number`): | Vector | NDArray The base.
- **exponent** (`Number`): The exponent.

## Returns

| Vector | NDArray The result of base ^ exponent.

## Examples

```t
pow(2, 3)
-- Returns: 8.0
```
