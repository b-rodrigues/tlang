# cummin

Cumulative Minimum

Calculates the cumulative minimum of a vector.

## Parameters

- **x** (`Vector`): The input numeric vector.

## Returns

The cumulative minimum.

## Examples

```t
cummin([3, 2, 4, 1])
-- Returns: [3, 2, 2, 1]
```
