# sd

Standard Deviation

Calculates the sample standard deviation of a numeric vector.

## Parameters

- **x** (`Vector`): | List The numeric data.
- **na_rm** (`Bool`): (Optional) logical. Should missing values be removed? Default is false.

## Returns

The standard deviation.

## Examples

```t
sd([1, 2, 3, 4, 5])
-- Returns: 1.5811...
```
