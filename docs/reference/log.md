# log

Natural logarithm

Calculates the natural logarithm (base e) of x.

## Parameters

- **x** (`Number`): | Vector | NDArray The input value (must be positive).

## Returns

| Vector | NDArray The natural logarithm.

## Examples

```t
log(2.71828)
-- Returns: ~1.0
```
