# sqrt

Square root

Calculates the square root of x.

## Parameters

- **x** (`Number`): | Vector | NDArray The input value (must be non-negative).

- **na_ignore** (`Bool`): Whether to preserve NA values in inputs. Default is false.


## Returns

| Vector | NDArray The square root.

## Examples

```t
sqrt(16)
-- Returns = 4.0
```
