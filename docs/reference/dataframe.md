# dataframe

Create a DataFrame

Constructs a DataFrame from a list of dictionaries (rows).

## Parameters

- **rows** (`List[Dict]`): The data rows.

## Returns

The created DataFrame.

## Examples

```t
df = dataframe([
{"a": 1, "b": 2},
{"a": 3, "b": 4}
])
```
