# cumsum

Cumulative Sum

Calculates the cumulative sum of a vector.

## Parameters

- **x** (`Vector`): The input numeric vector.

## Returns:

Returns: The cumulative sum.

## Examples

```t
cumsum([1, 2, 3])
-- Returns = [1, 3, 6]
```
