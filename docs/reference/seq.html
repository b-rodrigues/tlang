T Function Reference - seq

seq

Generate a sequence of integers

Creates a list of integers from start to end (inclusive).

Parameters

  • start (Int): Starting value.
  • end (Int): Ending value.

Returns

List of integers.

Examples

seq(1, 5)
-- Returns: [1, 2, 3, 4, 5]
Best viewed with a sense of curiosity.
