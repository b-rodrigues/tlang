# min

Minimum value

Returns the minimum value in a vector or list.

## Parameters

- **x** (`Vector`): | List The numeric data.

- **na_rm** (`Bool`): Whether to remove NA values. Default is false.


## Returns

The minimum value.

## Examples

```t
min([1, 2, 3])
-- Returns = 1.0
```
