# make_datetime

Construct a Datetime value

Builds a Datetime value from named date, time, and timezone components.

## Parameters

- **year** (`Int`): = 1970

- **month** (`Int`): = 1

- **day** (`Int`): = 1

- **hour** (`Int`): = 0

- **min** (`Int`): = 0

- **sec** (`Int`): = 0

- **tz** (`String`): = "UTC"


## Returns

The constructed datetime.
