# max

Maximum value

Returns the maximum value in a vector or list.

## Parameters

- **x** (`Vector`): | List The numeric data.

- **na_rm** (`Bool`): Whether to remove NA values. Default is false.


## Returns

The maximum value.

## Examples

```t
max([1, 2, 3])
-- Returns = 3.0
```
