# cummax

Cumulative Maximum

Calculates the cumulative maximum of a vector.

## Parameters

- **x** (`Vector`): The input numeric vector.

## Returns

The cumulative maximum.

## Examples

```t
cummax([1, 3, 2, 4])
-- Returns: [1, 3, 3, 4]
```
