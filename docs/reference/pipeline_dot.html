T Function Reference - pipeline_dot

pipeline_dot

Export Pipeline as DOT Graph

Returns a string containing a Graphviz DOT representation of the pipeline’s dependency graph, suitable for visualization.

Parameters

  • p (Pipeline): The pipeline.

Returns

A DOT graph string.

Examples

pipeline_dot(p)

See Also

pipeline_edges, pipeline_print

Best viewed with a sense of curiosity.
