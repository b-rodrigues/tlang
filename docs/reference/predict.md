# predict

Linear Model Prediction

Calculates predicted values for a linear model object. Supports models generated by lm() or imported via PMML and ONNX. Also supports PMML decision trees, random forests, and XGBoost/LightGBM.  For ONNX models, features are automatically resolved via model metadata or by matching numeric columns in order.

## Parameters

- **data** (`DataFrame`): The new data used for prediction.

- **model** (`Model`): The linear / ensemble / ONNX model object.


## Returns

The predicted values.

## Examples

```t
-- ONNX Prediction Example
model = t_read_onnx("model.onnx")
df = read_csv("data.csv")
preds = predict(df, model)

```
