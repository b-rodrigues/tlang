# predict

Linear Model Prediction

Calculates predicted values for a linear model object. Supports models generated by lm() or imported via PMML. Also supports PMML decision trees and random forests.

## Parameters

- **data** (`DataFrame`): The new data used for prediction.

- **model** (`Model`): The linear model object.


## Returns

The predicted values.

## Examples

```t
model = lm(mpg ~ wt + hp, data = mtcars)
preds = predict(mtcars, model)
```
