# make_period

Create a period value

Builds a period from named year, month, day, hour, minute, and second components.

## Parameters

- **years** (`Int`): (Optional)

- **months** (`Int`): (Optional)

- **days** (`Int`): (Optional)

- **hours** (`Int`): (Optional)

- **minutes** (`Int`): (Optional)

- **seconds** (`Int`): (Optional)


## Returns

The constructed period.
