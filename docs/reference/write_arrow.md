# write_arrow

Write Arrow IPC file

Writes a DataFrame to an Apache Arrow IPC (Feather v2) file.

## Parameters

- **df** (`DataFrame`): The DataFrame to write.

- **path** (`String`): The output file path.


## Returns



## Examples

```t
write_arrow(df, "data.arrow")
```
