# exp

Exponential function

Calculates e raised to the power of x.

## Parameters

- **x** (`Number`): | Vector | NDArray The input value.

- **na_ignore** (`Bool`): Whether to preserve NA values in inputs. Default is false.


## Returns

| Vector | NDArray The exponential.

## Examples

```t
exp(1)
-- Returns = 2.71828...
```
